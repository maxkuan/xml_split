# SPLIT PATENT IN XML

## USAGE

| Parameter     | Value                 |
| ------------- | --------------------- |
| -f, —filename | Source file           |
| -o, —output   | Folder to output file |
| -c, —count    | Null                  |



### Split file to folder

```shell
sh xml_split.sh -f filename -o output_folder [-c]
```

### Count patents in file

```shell
sh xml_split.sh -f filename -c
```
